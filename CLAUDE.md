# Azure SDK for Java — Claude Guidelines

## Hard Rules

### Never edit any storage model/header files

**Never** make any changes to files under `sdk/storage/` that define models or response headers. This includes both:

- AutoRest-generated files (marked with `// Code generated by Microsoft (R) AutoRest Code Generator.`) such as `BlobsDownloadHeaders.java` and anything under `*/implementation/models/`
- Hand-maintained model wrappers such as `BlobDownloadHeaders.java` and any other `*Headers.java` / `*Models.java` file under `sdk/storage/azure-storage-blob/src/main/java/com/azure/storage/blob/models/`

All edits to these files must go through the proper spec/model pipeline, not direct file changes.
