# Azure Core HTTP Common Test shared library for Java

Azure Core HTTP Common Test shared library for Java is a shared library for testing Azure Core HTTP clients. The test
classes defined by this library are used to validate the implementation of Azure Core HTTP clients.

## Getting started

### Prerequisites

- A [Java Development Kit (JDK)][jdk_link], version 8 or later.
  - Here are details about [Java 8 client compatibility with Azure Certificate Authority][java8_client_compatibility].

### Include the package

[//]: # ({x-version-update-start;com.azure:azure-core-http-test-common;current})
```xml
<dependencies>
  <dependency>
    <groupId>com.azure</groupId>
    <artifactId>azure-core-http-test-common</artifactId>
    <version>1.0.0-beta.1</version>
  </dependency>
</dependencies>
```
