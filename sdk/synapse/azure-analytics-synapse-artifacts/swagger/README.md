# Azure Synapse Analytics Orchestration Service for Java

> see https://aka.ms/autorest

### Setup

```ps
npm install -g autorest
```

### Generation
```ps
cd <swagger-folder>
autorest
```

### Code generation settings
```yaml
branch: main
repo: https://github.com/Azure/azure-rest-api-specs/blob/$(branch)
```
