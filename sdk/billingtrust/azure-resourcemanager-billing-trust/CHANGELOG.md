# Release History

## 1.0.0-beta.1 (2026-07-01)

- Azure Resource Manager BillingTrust client library for Java. This package contains Microsoft Azure SDK for BillingTrust Management SDK. Microsoft.BillingTrust resource provider. Provides Assessment and Rule resources used to evaluate trust attributes (such as education-domain qualification and business verification) of a billing account. Package api-version 2026-03-17-preview. For documentation on how to use this package, please see [Azure Management Libraries for Java](https://aka.ms/azsdk/java/mgmt).
