# Release History

## 1.0.0-beta.1 (2026-02-04)

- Azure Resource Manager Container Registry Tasks client library for Java. This package contains Microsoft Azure SDK for Container Registry Tasks Management SDK. The Microsoft Azure Container Registry management API provides create, read, update, and delete functionality for Azure Container Registry resources including registries, replications, webhooks, tasks, runs, and other registry components. Package api-version 2025-03-01-preview. For documentation on how to use this package, please see [Azure Management Libraries for Java](https://aka.ms/azsdk/java/mgmt).
### Features Added

- Initial release for the azure-resourcemanager-containerregistry-tasks Java SDK.
