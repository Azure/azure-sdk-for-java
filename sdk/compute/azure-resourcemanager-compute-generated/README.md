# Azure Resource Manager Compute client library for Java

Azure Resource Manager Compute client library for Java.

This package contains Microsoft Azure SDK for Compute Management SDK. Compute Client. Package tag package-2020-10-01-preview.

## Getting started

### Prerequisites

- Java Development Kit (JDK) with version 8 or above

### Adding the package to your product

```xml
<dependency>
    <groupId>com.azure.resourcemanager</groupId>
    <artifactId>azure-resourcemanager-compute-generated</artifactId>
    <version>1.0.0-beta.1</version>
</dependency>
```
